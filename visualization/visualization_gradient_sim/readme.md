# Gradient Cosine Similarity Visualization

To generate the gradient cosine similarity visualization, please follow the steps below:

## Steps

1. **Run `process.sh`**  
   This script will generate the required `.npy` file.

2. **Run `visualization.py`**  
   Use `visualization.sh` to execute the script and produce the final visualization.

## Notes
- Remember to update the file paths before running the scripts.
- We also provide a pre-generated `.npy` file at the following link:
  *(https://drive.google.com/drive/folders/1ji98IzkY-uXv8aOy8BnowpDhgNERis1m?usp=drive_link)*
